## Descripción
Este proyecto es un portafolio personal responsive diseñado para practicar la estructura semántica de HTML5, estilos CSS3 y funcionalidades básicas de JavaScript.

## Tecnologías utilizadas
* **HTML5:** Uso de etiquetas semánticas (`header`, `main`, `article`, `footer`).
* **CSS3:** Diseño responsivo (Mobile First) y Flexbox.
* **JavaScript:** Validación de formularios y manipulación del DOM.
* **VS Code:** Editor de código.

## Estructura del proyecto
El proyecto sigue la estructura requerida:
* `/web`: Carpeta principal del sitio.
* `/web/css`: Hoja de estilos.
* `/web/js`: Scripts de funcionalidad.
* `/web/img`: Imágenes y recursos.
