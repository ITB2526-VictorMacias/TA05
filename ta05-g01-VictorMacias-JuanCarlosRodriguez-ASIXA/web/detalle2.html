Proyecto: Automatización de Usuarios

Fecha: Noviembre 2025 | Asignatura: Sistemas Operativos

Descripción del Script

En el entorno de administración de sistemas, la eficiencia es clave. Este proyecto consistió en la creación de un script en Bash para automatizar la creación masiva de usuarios en un servidor Ubuntu Server.

El script lee un archivo `.csv` que contiene nombres, departamentos y contraseñas temporales. Automáticamente crea el usuario, le asigna su grupo correspondiente y configura una carpeta personal con los permisos adecuados (chmod/chown).

Fragmento de código destacado:

#!/bin/bash
# Bucle para leer el archivo de empleados
while IFS=, read -r nombre grupo; do
  if [ ! -d "/home/$nombre" ]; then
    useradd -m -g "$grupo" -s /bin/bash "$nombre"
    echo "Usuario $nombre creado con éxito."
  fi
done < empleados.txt

Retos superados:

  • Gestión de permisos de usuarios y grupos en Linux.
  • Validación de datos de entrada para evitar errores en la ejecución.
  • Automatización de tareas repetitivas mediante bucles.
