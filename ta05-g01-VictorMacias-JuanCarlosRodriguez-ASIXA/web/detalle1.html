Proyecto: Web Corporativa

Fecha: Octubre 2025 | Asignatura: Lenguajes de Marcas

Descripción detallada

Este proyecto consistió en el desarrollo integral de un sitio web para una PYME ficticia dedicada a la tecnología. El objetivo principal era crear una interfaz limpia, moderna y totalmente adaptativa (Responsive Design) que funcionara perfectamente en ordenadores de escritorio, tablets y teléfonos móviles.

Durante el desarrollo, me centré en la experiencia de usuario (UX), asegurando que la navegación fuera intuitiva. Utilicé la metodología "Mobile First", diseñando primero para pantallas pequeñas y escalando el contenido para pantallas grandes.


Tecnologías y Herramientas:

  • HTML5 Semántico: Uso correcto de etiquetas main, section, article y nav para mejorar el SEO.
  • CSS3: Implementación de Flexbox para la maquetación y Media Queries para la adaptabilidad.
  • VS Code: Entorno de desarrollo con extensiones como Live Server.
  • Git/GitHub: Control de versiones para gestionar los cambios en el código.
